
# 📊 Footwear Sales Insights: Global Analysis (2018-2026)

## 📝 Descrição do Projeto
Este projeto apresenta uma análise end-to-end do mercado global de calçados esportivos, cobrindo desde o faturamento bruto até o comportamento detalhado do consumidor. O objetivo foi transformar dados brutos em inteligência de negócio para suporte à decisão.

## 🚀 Tecnologias Utilizadas
* **Power BI**: ETL, Modelagem de Dados e Data Viz.
* **DAX**: Cálculos de Inteligência de Tempo (YoY) e Medidas de Performance.
* **Excel**: Base de dados original.

## 📈 Principais Insights
* **Liderança de Mercado:** A ASICS consolidou-se como a marca de maior faturamento acumulado.
* **Produto Estrela:** O Model-863 é o principal gerador de receita (R$ 17,8 Mil).
* **Comportamento de Compra:** Transferências bancárias são o método preferido, representando R$ 2,3 Mi do volume total.
